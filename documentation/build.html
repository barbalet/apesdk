Noble Ape Simulation

Overview

The Noble Ape Simulation has been built with a variety of development environments including CodeWarrior, Visual Studio and XCode. As a majority of the development environments used to build the Simulation have not included traditional make files, the Simulation source is not packaged with these make files. These development environment project files (with the exception of CodeWarrior) are available through:

   https://gitlab.com/barbalet/nobleape_ui - the general and command line builds of the Simulation
   https://gitlab.com/barbalet/nobleape_client - the server and client builds of the Simulation

Command Line Environment

There are a number of command line versions. These should build for Mac and Linux:

   build.sh - builds the nalongterm command line version of the Noble Ape Simulation that allows for a variety of executions and analysis

   test.sh - runs a number of unit tests on the overall simulation including the Noble Toolkit specific tests
   test_object_file.sh - runs tests against Noble Ape JSON files
   test_object_string.sh - runs tests against Noble Ape JSON strings
   test_object.sh - runs tests against Noble Ape JSON parsing and analysis

   doc.sh - generates documentation from source code

   server.sh - builds the Noble Ape Server command line
